What is XML?

  • XML stands for EXtensible Markup Language
  • XML is a markup language much like HTML
  • XLM was designed to describe data
  • XLM tags are not predefined.You must define your own tags
  • XLM uses a Document Type Definition (DTD) or an XLM Schema to describe the data
  • XLM with a DTD or XLM Schema is designed to be self-descriptive

The main difference between XML and HTML

XLM was designed to carry data.

XML is not a replacement for HTML.

XML i not a replacement for TTML.
XML and html were designed with different goals :

XML was designed to describe data and to focus on what data is.
HTML was designed to display data and focus on how data looks.

XML is about displaying information,While XML is about describing information.

XML does DO anything

XML was not designed to DO anything.
